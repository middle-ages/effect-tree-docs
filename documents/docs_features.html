    Tree Operations

    1. A generic tree data structure and all the basic functions required to manipulate immutable instances.
    2. A zipper for navigation and efficient modification.
    3. Measure node counts/heights/depth/degree, crop by depth, filter trees, unfold level trees, select subtrees, zip/unzip trees, and traverse in any order.
    4. Annotate nodes with their level index, for example 2.3.11.1, encoding their path from root.
    5. Numeric and boolean folds on nodes. For example sum all nodes, find maximum node value for some given order, or check if all nodes are true. All operations short-circuit whenever possible.
    6. Folds can operate on entire trees, replace nodes, annotate nodes, and are composable to tuples and structs. This allows you to count nodes, annotate with depth, and crop depth, for example, in a single fused iteration over the tree.
